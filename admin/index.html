Dicas:

• Artigos de URL única podem ser importados como Rascunho, Publicado ou Destaque.
• Imagens externas são referenciadas por URL (não são salvas localmente).

Gerar Páginas HTML de Artigos

Gere arquivos HTML estáticos para cada notícia publicada. Os arquivos serão criados com URLs amigáveis (ex: corinthians-campeao-supercopa.html) e terão o visual completo do site, incluindo imagem importada.

Como funciona:
• Cada notícia publicada gera um arquivo .html independente
• O arquivo usa o layout e CSS do portal
• A imagem da notícia é embutida automaticamente
• Links internos apontam para os slugs corretos
Geração automática via Node.js:
1. Clique em "Exportar data/news.json" para salvar suas notícias
2. Coloque o arquivo na pasta data/news.json do projeto
3. Execute no terminal: node generate.js
4. Os arquivos HTML serão criados na pasta artigos/
